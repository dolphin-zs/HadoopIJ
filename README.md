HadoopIJ
========

Secondary Development based on HIPI (Hadoop Image Processing Interface) and ImageJ

Use maven to build.

Use hadoop 2.x jars.

Usage:

1. `mvn package`

2. `cd target/classes`

3. `java -cp ../Hipi2ndDev-0.0.1-jar-with-dependencies.jar dev.zs.bundle.BuildHipiImageBundle <input-path> <output-name>.hib`
